#### Installation

Clone the repository:

```git clone git@github.com:ArtemyAA/python-project-83.git```

#### Usage

Install dependencies using bash script and also set the database:

```make build```

Install dependencies with Poetry:

```make pack-install```

Start the server:

```make start```
